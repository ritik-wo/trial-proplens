# Sales Admin Panel

Short guide to run the app and the tests.

## Prereqs
- Node 18+
- npm

## Install
```bash
npm install
```

## Run the app (dev)
```bash
npm run dev
```
The app starts at http://localhost:3000

## Lint & type-check (optional)
```bash
npm run lint
npm run typecheck
```

## E2E tests (Cypress)
```bash
# headless
npm run test:e2e
# headed runner
npm run test:e2e:headed
```
